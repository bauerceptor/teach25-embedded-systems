# Extra Questions: Computer Security

## Part A: Multiple Choice Questions

### 1. Which of the following is a characteristic of a worm?
a) Attaches to other files to spread
b) Self-replicates across networks
c) Only affects single computers
d) Requires user action to activate

### 2. The CIA triad in computer security refers to:
a) Central Intelligence Agency protocols
b) Confidentiality, Integrity, Availability
c) Computer Information Assurance
d) Cyber Intelligence Analysis

### 3. Which type of information system is best suited for analyzing large amounts of historical data?
a) Expert System
b) Office Automation System
c) Data Warehouse
d) Geographic Information System

### 4. An attack vector is:
a) A type of malware
b) A security vulnerability
c) A method used to gain unauthorized access
d) A firewall configuration

### 5. Which of the following is NOT a component of information systems?
a) Hardware
b) Human Resources
c) Marketing
d) Telecommunications

### 6. Phishing attacks primarily rely on:
a) Software vulnerabilities
b) Network intrusions
c) Social engineering
d) Hardware manipulation

a) Level of technical skill
b) Type of tools used
c) Intent and legality of actions
d) Educational background

### 8. Which security principle ensures that data has not been modified?
a) Confidentiality
b) Availability

c) Network monitoring devices

## Part B: Short Answer Questions

### 16. **Security Fundamentals**
Explain the relationship between risk, threat, and vulnerability with a practical example.

### 17. **Information System Types**
Compare and contrast Expert Systems and Geographic Information Systems, including their typical applications.

### 18. **Security Principles**
Describe how the principle of accountability is implemented in computer systems and why it's important.

### 19. **Malware Analysis**
Explain how a botnet operates and why it's considered particularly dangerous for organizations.

### 20. **Authentication Methods**
List and describe three different types of authentication methods, including their strengths and weaknesses.

### 21. **Network Security**
Explain why both antivirus software and firewalls are necessary for comprehensive computer protection.

### 22. **Data Protection**
Describe the difference between backup and encryption in terms of data protection goals.

### 23. **Security Assessment**
Outline the five basic steps for computer protection and explain why each step is important.

### 24. **Incident Response**
Describe what actions should be taken immediately after discovering a malware infection.

### 25. **Security Policies**
Explain why user education is considered a critical component of computer security.

## Part C: Practical Application Questions

### 26. **Windows Security Configuration**
You are setting up a new Windows 10 computer for a small business. Describe the complete security configuration process including antivirus installation, firewall setup, and backup configuration.

### 27. **Security Threat Analysis**
A company has experienced the following incidents in the past month: employees receiving suspicious emails, slow network performance, and unauthorized login attempts. Analyze these symptoms and recommend appropriate security measures.

### 28. **Information System Design**
Design a simple information system for a library that includes all five components. Explain how each component contributes to the system's functionality and identify potential security vulnerabilities.

### 29. **Firewall Rule Configuration**
Create a set of firewall rules for a home office computer that:
- Blocks all incoming connections except for specific applications
- Allows outbound web browsing and email
- Logs all blocked connection attempts
- Permits file sharing on the local network only

### 30. **Incident Investigation**
You discover that several files on your computer have been encrypted by malware (ransomware). Describe the step-by-step process for handling this security incident, including immediate actions and recovery procedures.

### 31. **Security Audit**
Perform a basic security assessment of a computer system. Create a checklist of items to verify and explain how each item contributes to overall system security.

### 32. **Data Recovery Scenario**
A user accidentally deleted important business documents and the recycle bin has been emptied. Describe multiple recovery options using Windows built-in tools and explain the likelihood of successful recovery for each method.

### 33. **Network Security Planning**
Design a network security plan for a small office with 10 computers. Include recommendations for firewalls, antivirus software, backup strategies, and user training.

### 34. **Encryption Implementation**
Explain how to implement file encryption on a Windows system for protecting sensitive customer data. Include steps for individual files, folders, and considerations for key management.

### 35. **Security Monitoring**
Set up a system for monitoring and logging security events on a Windows computer. Describe what events should be logged, how to configure logging, and how to analyze the logs for security issues.

## Part D: Case Study Questions

### 36. **Corporate Security Breach**
A medium-sized company discovers that customer credit card information has been stolen from their database. The investigation reveals that the breach occurred through an employee's computer that was infected with malware after opening a phishing email.

**Questions:**
a) Identify all the security principles that were violated
b) Classify the types of losses the company might face
c) Recommend preventive measures to avoid similar incidents
d) Describe the immediate response actions needed

### 37. **Remote Work Security**
Due to recent events, a company needs to enable 200 employees to work from home using their personal computers. Design a comprehensive security strategy that addresses:
- Secure access to company data
- Protection of personal and company information
- Network security considerations
- User training requirements

### 38. **IoT Device Security**
A smart home system includes connected cameras, door locks, thermostats, and lighting controls. Analyze the security challenges specific to these IoT devices and recommend security measures for:
- Device authentication
- Data encryption
- Network segmentation
- Regular security updates

### 39. **Educational Institution Security**
A university wants to improve its computer security across campus labs, student dormitories, and administrative offices. Develop a security framework that addresses:
- Different user groups (students, faculty, staff)
- Various types of data (academic records, research data, personal information)
- Mixed device environments (university-owned and personal devices)
- Compliance with educational privacy regulations

### 40. **Small Business Recovery**
A small business (15 employees) has been hit by ransomware that encrypted most of their files. They have some backups but they're two weeks old. Create a recovery plan that includes:
- Immediate containment actions
- Data recovery strategies
- Business continuity measures
- Long-term security improvements

## Assessment Guidelines

### For Instructors:
1. **Multiple Choice (40%)** - Test basic concept understanding
2. **Short Answer (30%)** - Evaluate explanation and analysis skills
3. **Practical Application (20%)** - Assess implementation knowledge
4. **Case Studies (10%)** - Test comprehensive problem-solving

### Study Tips for Students:
1. **Understand relationships** between security concepts
2. **Practice hands-on** Windows security configurations
3. **Analyze real-world** security scenarios
4. **Review current** security threat landscapes
5. **Focus on practical** implementation skills
