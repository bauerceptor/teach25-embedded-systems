## Answer the following questions

**1. Define Computer Security.**

Computer security means dealing with the security of a standalone computer's software and hardware. One of the most important aspects of computer security is keeping stand-alone machines updated and properly patched.

**2. Differentiate between Computer Security and Cyber Security.**

Computer Security focuses on protecting standalone computers and individual systems, covering protection of software, hardware, and data on single machines. Cyber Security involves the security of individual computers plus the network they belong to and the intercommunication between all the computers in an organization. It encompasses everything that is connected and deals with network-based threats and distributed attacks.

**3. Define threats, attack vectors and vulnerability.**

**Threat:** A process that magnifies the likelihood of a negative event, such as the exploit of a vulnerability. Today's cybersecurity landscape includes potential threats from malware, ransomware, and specially targeted hacker attacks.

**Attack Vector:** A path or means by which an attacker or hacker can gain access to a computer or network server in order to deliver a payload or malicious outcome. Common attack vectors include viruses, malware, email attachments, webpages, pop-up windows, instant messages, chatrooms and deception.

**Vulnerability:** A weakness in your infrastructure, networks or applications that potentially exposes you to threats. Organizations can have thousands, often millions of vulnerabilities.

**4. What are Information systems?**

Information systems incorporate computers to help manage data and achieve business objectives, they do not necessarily have to include computers. There are different types of information systems that can serve a variety of purposes depending on an organization's needs, including data warehouses, enterprise systems, expert systems, geographic information systems, and office automation systems.

**5. Describe assets of an Information system.**

Every information system includes several key components: hardware, software, telecommunications, people and data. Hardware refers to physical pieces, software is the programming that controls the system, telecommunication transmits information through the system, humans manage and interact with the system, and data is information stored within and processed by the system.

**6. Define hacker, cracker and security expert.**

**Hacker:** People who use their knowledge for a good purpose and do not damage the data. They possess advanced knowledge of computer systems and programming languages and work for organizations to improve their network and solve issues. Hacking is ethical and hackers have ethical certificates.

**Cracker:** Someone who breaks into the system with a malicious purpose and damages data intentionally. They might not necessarily be skilled and well-versed with computing knowledge. Cracking is illegal and unethical, and crackers do not possess any certificates.

**Security Expert:** A professional who specializes in protecting computer systems and networks from threats, implementing security measures, and responding to security incidents.

**7. Define confidentiality, integrity and authenticity.**

**Confidentiality:** Protecting information from being exposed to an unauthorized party due to a data breach or insider threat. It covers a spectrum of access controls and measures that protect information from getting misused by any unauthorized access. Information security basics include encryption, passwords, two-factor authentication, and biometric verification.

**Integrity:** The accuracy and completeness of data. Security controls focused on integrity prevent data from being modified or misused by unauthorized parties. It involves maintaining consistency and trustworthiness of data over its entire life cycle. Controls include encryption, user access controls, version control, backup and recovery procedures, and error detection software.

**Authenticity:** The process of identifying users that request access to a system, network, or device. Access control determines user identity according to credentials like username and password. Other authentication technologies like biometrics and authentication apps are also used to authenticate user identity.

**8. Describe Encryption and Decryption.**

**Encryption:** The process of translating plain text data (plaintext) into something that appears to be random and meaningless (ciphertext). It uses algorithms and keys to scramble information and protects confidentiality of sensitive data.

**Decryption:** The process of converting ciphertext back into plaintext. It requires the proper key and algorithm to restore access to original information.

**9. Describe Malware.**

Malware refers to malicious software programs designed to harm, exploit, or gain unauthorized access to computer systems. Common types include viruses (replicate and infect files), worms (copy themselves across networks automatically), keyloggers (record keystrokes to steal credentials), botnets (networks of compromised computers), and phishing (fraudulent attempts to steal sensitive information).

**10. Differentiate between Firewall and Anti-virus.**

**Firewall:** Controls network traffic in the system by acting as barriers for incoming traffic. It monitors incoming and outgoing network traffic and decides whether to allow or block specific traffic based on defined security rules. Firewalls protect against network-based threats and unauthorized access attempts.

**Anti-virus:** Protects systems against internal attacks by perceiving or spotting malicious files and viruses. Its primary function is to scan, spot, and inhibit distrustful files and software from getting into the system. Anti-virus software provides real-time protection and removes malicious threats.

## Practical Tasks

**1. Install antimalware software on Windows.**

Download reputable antimalware software from official website, run installer with administrator privileges, follow setup wizard accepting license terms, configure real-time protection settings, complete installation and restart if required. Enable automatic updates for virus definitions.

**2. Perform a malware scan on Windows.**

Open antimalware software, verify all security updates are downloaded, execute a full system scan covering hard drives, removable media, system memory and email, read result reports for further action, follow prompts to remove or quarantine detected threats.

**3. Install and configure a firewall on Windows.**

Access Windows Defender Firewall through Control Panel, enable firewall for different network profiles (domain, private, public), configure inbound and outbound rules for specific applications and ports, set notification preferences for blocked programs, test configuration to verify proper operation.

**4. Configure firewall to create log files.**

In Windows Defender Firewall advanced settings, enable logging for dropped packets and successful connections, set log file location and size limits, configure log rotation policies, specify logging settings for troubleshooting purposes.

**5. Perform a system restore on Windows.**

Access System Restore through Control Panel > System and Security > System, select "System Restore", choose restore point from available dates, review affected programs and drivers, confirm restore operation, allow system to restart and complete restoration process.
