# Chapter 2: Electric Circuits

## Overview
This chapter builds upon basic electrical theory to cover comprehensive circuit analysis techniques and advanced circuit concepts used in electronic system design.

## Learning Objectives
- Master advanced circuit analysis techniques
- Understand network theorems and their applications
- Learn about AC circuit analysis and phasor diagrams
- Explore frequency response and filtering concepts
- Apply circuit analysis to practical electronic systems

## Key Topics Covered
- Advanced circuit analysis methods
- Network theorems (Thevenin, Norton, Superposition)
- Node voltage and mesh current analysis
- AC circuit analysis and phasor representation
- Impedance and admittance concepts
- Frequency response and Bode plots
- Filter circuits (low-pass, high-pass, band-pass)
- Resonance in AC circuits
- Power factor and reactive power
- Circuit simulation tools and techniques

## Prerequisites
- Basic Electrical Theory (Chapter 1)
- Solid understanding of Ohm's law and basic circuits
- Mathematical skills in algebra and trigonometry

## Laboratory Activities
- Complex circuit analysis and verification
- AC circuit measurements with oscilloscopes
- Filter circuit design and testing
- Network theorem verification
- Circuit simulation using SPICE tools
